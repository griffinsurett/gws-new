Adding content to Greastro is so simple:

1. **Choose a collection**: Navigate to `src/content/[collection-name]`
2. **Create an MDX file**: Add a new `.mdx` file with frontmatter
3. **Add your content**: Write your content in Markdown below the frontmatter
4. **View your content**: The page is automatically generated
